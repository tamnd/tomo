Three short pages: how tomo thinks about an agent (a local daemon behind your chat apps, acting through tools that pass a policy gate first, pointed at whatever model you like), how to install the single binary, and a guided first run that ends with a working agent you can talk to in the terminal and in a browser.

tomo (友, "companion") runs on your hardware.
Your provider key, your history, and your memory stay on the box, and the only thing that leaves is the model call you would have made anyway.

Start with the [introduction](/getting-started/introduction/) for the shape of it, then [install](/getting-started/installation/) and follow the [quick start](/getting-started/quick-start/).
