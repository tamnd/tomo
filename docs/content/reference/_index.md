The precise layer under the guides.

The [guides](/guides/) explain what tomo does and why.
This reference lists exactly what it accepts.

- The [CLI reference](/reference/cli/) covers every command and subcommand, its arguments, and its real flags.
- The [config file](/reference/config-file/) reference maps every key in `~/.tomo/config.yaml`, its default, and what it controls.

Nothing here is invented.
If a flag or a key is not on these pages, tomo does not read it.
