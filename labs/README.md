# labs

Reproducible review evidence organized after `tamnd/aki/labs`.
Each numbered directory isolates one question, documents the threat and expected invariant, and contains an executable Go test.
The labs exercise tomo's real public packages when the question crosses an integration boundary.

Run every lab with:

```sh
go test ./labs/...
```
